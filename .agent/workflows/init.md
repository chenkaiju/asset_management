為了確保在不同裝置間開發的連貫性，請執行以下步驟來初始化對話背景：

1. **優先讀取根目錄文件**：
   讀取並分析專案根目錄下的以下文件，以了解目前的進度與計畫：
   - [walkthrough.md](file:///d:/antigravity/asset_management/walkthrough.md)
   - [task.md](file:///d:/antigravity/asset_management/task.md)
   - [implementation_plan.md](file:///d:/antigravity/asset_management/implementation_plan.md)

2. **同步至 Brain (Artifacts)**：
   將上述文件的內容同步到當前對話的 Brain (Artifacts) 目錄中，以便在後續開發中持續更新。

3. **確認開發目標**：
   根據 `task.md` 中尚未完成的項目，詢問使用者接下來的開發重點。

4. **檢查 Git 狀態**：
   執行 `git status` 確保本地端代碼與 Git 紀錄同步。
